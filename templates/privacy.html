Privacy Policy

LikeTube AI respects your privacy and is committed to protecting your personal data.

Google User Data Access

LikeTube AI allows users to sign in with their Google account. After the user explicitly authorizes access through Google OAuth, the application uses the YouTube Data API with the following scope:
https://www.googleapis.com/auth/youtube.readonly

Data We Access

After authorization, the application may access the user's YouTube liked videos, including video titles, channels, thumbnails, duration, and other metadata provided by the YouTube Data API.

How We Use Data

The accessed YouTube data is used only to display and filter the user's liked videos inside the LikeTube AI interface. Users can search, organize, and filter their liked videos by title, channel, duration, and popularity.

Data Storage

LikeTube AI does not permanently store YouTube user data. The data retrieved from the YouTube API is used only during the active user session and is not stored in external databases.

Data Sharing

LikeTube AI does not sell, transfer, or share Google user data with third parties.

Revoking Access

Users may revoke access to their Google data at any time through their Google Account permissions page.

Compliance

LikeTube AI complies with the Google API Services User Data Policy and the YouTube API Services Terms of Service.
